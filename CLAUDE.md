Read [AGENTS.md](./AGENTS.md) at the repo root first. It is the canonical rules file for
RedShiftOS.

`AGENTS.md` holds the context-loading order, the rules for building games with this OS, the
rules for maintaining the OS itself, and the standing behavioral rules for any agent. This
repo is docs-only — there is no build or test step. Everything below AGENTS.md
(`FOUNDATION/`, `PROCESS/`) is the operating system; read `FOUNDATION/` before `PROCESS/`.
