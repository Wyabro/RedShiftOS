# 4 · Implementation

**Build it** — the working loop in motion.

- **The loop:** Explore → Plan → Approve → Implement → Diff-review → Commit → Validate →
  Handoff (see `AGENTS.md`). Never go straight from request to code.

  folder).
- Route work by model, recover stuck sessions, and reuse prompts → `../../AI/`.
- **Gate:** both merge gates + the proof ladder (`FOUNDATION/Feature-Lifecycle.md`).
