# 5 · Playtesting

**Test with real players on real hardware.** Players > assumptions; test early, test often.

- Playtest the FTUE with dev shortcuts **off**, and sequence playtests deliberately — drain
  your own taste-tuning queue first, save fresh testers for first impressions (Lesson L-15).
- Judge feel and performance on target hardware, with levers you can hand a tester (Lesson
  L-06).
- Separate "intense" from "broken" — fix the bug, keep the spice (Lesson L-16).

Grows with: playtest checklists and bug/balance/fun-factor templates as games need them.
